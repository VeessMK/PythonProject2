# Домашняя работа!
Проект создан для выполнения домашней работы и практики в использовании [GitHub](https://github.com/)

## Проект:
Виджет для личного кабинета банка.
Бэкенд функции:
- Наложение масок(номер карты, номер счёта)
- Фильтры словарей-списков
- Сортировка словарей-списков
